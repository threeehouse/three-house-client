# Introduce
![trash-can-intro](https://github.com/threeehouse/trash-can/assets/47452547/e7c6f465-0e77-4406-ad19-aff288e609b4)

- 정든 쓰레기를 올리는 공간입니다.
- 불쾌할 수 있는 게시물을 방지하기 위해 기본적으로 블러처리 되어있습니다.
- 서버 사이드에서 데이터를 가져와 Pre-Rendering 하였습니다.
- 마음에 드는 쓰레기에 조용히 기도하기를 누를 수 있습니다.
- Interactive한 UI를 최대한 구현하려 노력했습니다.
- FE, BE를 혼자 작업한 Full Stack 프로젝트 입니다.
