- 인상깊은 쓰레기를 올리는 공간입니다.
- 불쾌할 수 있는 게시물을 방지하기 위해 기본적으로 블러처리 되어있습니다.
- 비록 쓰레기지만 검색되길 원해서 Server Side Rendering을 구현하였습니다.
- 마음에 드는 쓰레기에 조용히 좋아요를 누를 수 있습니다.
- Interactive한 UI를 최대한 구현하려 노력했습니다.
- FE, BE를 혼자 작업한 Full Stack 프로젝트 입니다.
- 아직 완성되지 않은 작업물 입니다.
